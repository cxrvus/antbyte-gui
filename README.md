# ANTBYTE GUI

Native Graphical User Interface for [AntByte](https://github.com/cxrvus/antbyte)

## Notes

- Also requires `antbyte` to be installed.
- `antbyte-gui` needs to have the same parent directory as `antbyte`
- if `antbyte-gui` is added to `PATH`, it can be used as a drop-in replacement for `antbyte` 

## Controls

- ctrl + scroll to zoom
- click and drag to pan
